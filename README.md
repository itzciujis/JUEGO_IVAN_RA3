# JUEGO_IVAN_RA3
Es un juego echo en java el cual se basa en sobrevivir un total de 7 dias en los cuales habras de controlar la comida, el agua y la vida evitando que cualquiera de esos valores llegue a 0 para no morir una vez llegues a los 7 dias sobrevividos habras completado el juego y habras ganado la partida
